# CodeAlpha_Calculator
A basic web-based calculator built using HTML, CSS, and JavaScript. It performs standard arithmetic operations with a clean and responsive UI.

# Features
-Basic operations: Addition (+), Subtraction (-), Multiplication (*), Division (/)
-Clear screen (C)
-Delete last digit (DEL)
-Decimal support (.)
-Simple and responsive UI
-Lightweight and fast
